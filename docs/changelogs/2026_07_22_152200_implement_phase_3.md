# Implementasi Fase 3 - Projects Showcase & Refactoring Theme

## Dikerjakan
- **Refactoring Dual Theme**: Menghapus fitur tema "Logikraf" dari `globals.css`, `brand-provider.tsx`, dan navigasi (`Navbar.tsx`) sesuai instruksi. Website kini menggunakan standar satu identitas (*Personal*) dengan mode *Light* dan *Dark* standar.
- **Projects Page (`/projects`)**: Membangun halaman daftar proyek (Grid Layout) yang membaca data dari modul lokal CMS (`projects.json`). Halaman menggunakan teknik grid responsif (`grid-cols-1 md:grid-cols-2 lg:grid-cols-3`) agar optimal untuk *mobile*.
- **Project Detail Page (`/projects/[slug]`)**: Membuat halaman rute dinamis (dinamis statis tersalin via `generateStaticParams`) untuk menampilkan studi kasus. Menampilkan parameter `slug`, *tech stack badges*, ringkasan masalah/solusi, serta tombol eksternal ke situs demonstrasi.
- **Mobile Friendly Checks**: Memastikan *padding* dan *stacking* (menggunakan `flex-col` pada layar kecil dan *grid* responsif) berlaku untuk halaman-halaman yang baru dikerjakan di Fase 3 ini.

## Diasumsikan
- URL gambar untuk proyek sementara tidak saya validasi ekstensi fisiknya karena file asli belum ada di `/public/images/projects/`, jadi saya menggunakan placeholder teks besar untuk representasi gambarnya agar lebih aman dari kerusakan *build* terkait *missing images*. 

## Belum Diverifikasi
- Pengalaman transisi antar-halaman (*page transition*) ketika menggunakan koneksi internet sangat lambat (3G throttling) di ponsel asli, walau dari segi kodenya sudah menggunakan *client routing* Next.js yang cukup optimal.
